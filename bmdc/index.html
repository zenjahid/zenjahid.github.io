BMDC API Documentation

This basic api enables user to authenticate the doctor with their registration number. Developed by zenjahid.

Version 1

Get Captcha

Request Type: GET

Request URL: https://bmdc-api.onrender.com/v1/get_captcha

Parameters: None

Response:

{
    "error": "no",
    "captcha_src": "https://verify.bmdc.org.bd/cpt/1720618813.933.jpg",
    "csrf_token_value": "a0b94f6fe222d26aa95d6a2cd0ff6e2d",
    "action_key_value": "vWb5HoQ6YwhR",
    "cookies": {
        "bmdckyc_csrf_cookie": "a0b94f6fe222d26aa95d6a2cd0ff6e2d"
    }
}
        
{
    "error": "yes"
}

Use the captcha source image and take the input from user. Pass the captcha input and doctor registration number from user along with the other responses from get request as parameter in post request

Get Doctor Info

Request Type: POST

Request URL: https://bmdc-api.onrender.com/v1/get_info

Parameters (as JSON):

{
    "bmdckyc_csrf_token": "xxx",
    "reg_ful_no": "xxx",
    "captcha_code": "xxx",
    "action_key": "xxx",
    "cookies": "xxx"
}
